Data Cleaning of a Global Layoff Dataset

Overview:

This project involved cleaning and preparing a global layoff dataset for analysis. The dataset contained 2,361 records across 9 key columns, detailing company layoffs across 60 countries. Key fields included the number of employees laid off, the percentage of workforce affected, the companyâ€™s location, industry, and growth stage, as well as the amount of funds raised. However, there were significant challenges with missing data, duplicate entries, unnecessary columns, and inconsistencies that required careful handling.

Data Summary:

  • The dataset featured 1,893 unique companies, with some companies appearing multiple times.
  • Key columns such as total_laid_off and percentage_laid_off contained substantial missing data (740 and 785 missing values, respectively).
  • There were 32 industries represented, with Finance being the most common, and 16 growth stages, predominantly Post-IPO.
  • The number of employees laid off ranged from 3 to 12,000, with an average of 238.

Data Cleaning Process:

  • Duplicate Handling: Identified and removed duplicate records based on key fields such as company name, industry, location, and layoff details using SQL queries with ROW_NUMBER().
  • Missing Values: Addressed missing data in crucial fields like total_laid_off and percentage_laid_off by removing or imputing values based on the analysis context.
  • Standardization: Cleaned up inconsistencies in text fields (e.g., company names and countries) and ensured uniform formatting for numerical fields such as layoff percentages and funds raised.
  • Removal of Unnecessary Columns and Rows: Removed columns and rows that were irrelevant or redundant, such as those containing irrelevant metadata or excessive null values, ensuring that the dataset only contained fields necessary for further analysis. This streamlined the dataset and improved its overall integrity.

Tools Used:

SQL: Data cleaning was performed using MySQL, where queries were written to handle duplicates, null values, standardize entries, and remove irrelevant data.
